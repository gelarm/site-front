ПАО «ФСК ЕЭС»

Автоматизация процесса мониторинга оборудования сети передачи данных

Задача проекта

Система мониторинга и управления сетью передачи данных единой технологической сети связи электроэнергетики

Решение

Централизованная система, работающая в автоматическом режиме. Построена на базе продуктов IBM Tivoli (Netcool/Omnibus, Netcool/Impact, Network Manager, Configuration Manager, Common Reporting) и SevOne NMS.

Функции

  • мониторинг производительности устройств сети передачи данных
  • оповещение персонала о проблемах по email, SMS, Telegram
  • отображение текущей и исторической информации по работоспособности сети передачи данных в виде настраиваемых виджетов и отчетов во встроенной системе бизнес-аналитики
  • событийный мониторинг
  • автоматическое обследование сети и топологической корреляции событий
  • обогащение событий данными из внешних систем (CMDB, система управления плановыми работами)
  • управление конфигурациями устройств сети передачи данных, анализ конфигураций на соответствие стандартов безопасности предприятия
  • автоматическое создание, управление и закрытие заявок и инцидентов в корпоративной системе управления инцидентами
